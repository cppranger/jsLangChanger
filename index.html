Welcome!

It's a little pet project, whose goal is to write a script in JavaScript, which switches the language on the page. Technologies such as HTML, CSS and JavaScript were used to create this project. Also used Bootstrap. If you click on the "Russian" button, the page text will change to Russian. If you click on the "German", the page text will change to German. If you click on the "English", the text does not change, since it is already in English. You will also hear a cat meow.
